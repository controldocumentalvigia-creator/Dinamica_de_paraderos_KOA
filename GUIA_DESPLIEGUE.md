# Guía de despliegue

## GitHub

La raíz debe verse así:

- app.py
- config.py
- requirements.txt
- Dinamica_de_paraderos_KOA_ES.xlsx
- .streamlit/
- core/
- dashboard/
- reports/
- ai/
- tests/
- README.md

## Streamlit Cloud

- Branch: main
- Main file path: app.py
- Python: 3.11 recomendado

Si la aplicación ya existía, use Manage app > Reboot app.
