# KOA Analytics V3 - Entrega completa

Aplicación modular de inteligencia operacional para KOA.

## Módulos incluidos

- Motor de carga y normalización.
- Validación matemática.
- Dashboard ejecutivo.
- Análisis mensual.
- Análisis semanal dentro de cada mes.
- Demanda.
- Frecuencia de paraderos.
- Combinaciones y tiempos.
- Puntualidad.
- Retorno real.
- Calidad operativa.
- Simulador de escenarios.
- Conclusiones y recomendaciones automáticas.
- Exportación a Excel, Word, PowerPoint y PDF.

## Despliegue en Streamlit Cloud

1. Suba todos los archivos y carpetas a la raíz del repositorio.
2. Configure `app.py` como Main file path.
3. Instale las dependencias desde `requirements.txt`.
4. Pulse Deploy o Reboot app.

## Regla importante

No elimine las carpetas `core`, `dashboard`, `reports` ni `ai`, porque `app.py` importa sus módulos.
